OMW Privacy Policy

Bugsnag

If you opt into automated crash reporting, when an error occurs some data will be collected from your device and transmitted to Bugsnag. This data includes but is not limited to:
  • Both automatically-generated and user OpenMW configuration files
  • List of installed mods
  • A unique device identifier
  • Amount of memory and disk
  • Battery level and charging state
  • Orientation of the device (portrait, landscape)
  • Whether access to location services is allowed by user or not
  • The type of network access available (e.g. wifi, cellular)
  • The device time
  • Manufacture, model and brand
  • Screen resolution and density
  • Information about the Android OS version & API level
  • Device locale (language/currency etc)
  • Whether the device is an emulator or not
  • The type of CPU (e.g. x86)
  • Whether the device is rooted or not
  • How long the application has been running for
  • How long the app has been in the foreground for
  • Whether the app is in the foreground or not
  • The screen name of the current foreground Activity
For more information, see Bugsnag privacy policy located at https://docs.bugsnag.com/legal/privacy-policy/
